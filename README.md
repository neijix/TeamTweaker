# TeamTweaker (Public Package Repo)

This repository contains the Chrome Web Store submission package for TeamTweaker.

## Included

- `TeamTweaker-v1.0.0.zip` (upload this file to Chrome Web Store)
- Extracted runtime extension files from that zip:
  - `manifest.json`
  - `background.js`
  - `content.js`
  - `styles/`
  - `icons/`
  - `popup/`
  - `sidepanel/`
  - `unread/`
- `PRIVACY_POLICY.md`

## Notes

- This public repository is intended for distribution/review artifacts.
- Development history and tooling remain in the private repository: `TeamTweaker-DEV`.
