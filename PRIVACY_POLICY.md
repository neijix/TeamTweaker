# Privacy Policy for TeamTweaker

Effective date: 2026-05-05

TeamTweaker is a browser extension for Microsoft Teams Web.

## What TeamTweaker does

TeamTweaker customizes the Microsoft Teams web interface (themes, chat wallpaper, readability options, unread/pinned utilities, and related visual settings).

## Data handling summary

- TeamTweaker does not sell user data.
- TeamTweaker does not transfer user data to third parties for advertising or unrelated purposes.
- TeamTweaker does not use user data for creditworthiness or lending purposes.
- TeamTweaker does not fetch and execute remote code; executable logic is packaged with the extension.

## What data may be processed

To provide extension features, TeamTweaker may process:

- Website content displayed in supported Microsoft Teams web pages.
- Conversation-related page content needed for unread/pinned conversation utilities.
- User-provided wallpaper image data selected locally by the user.

## Storage locations

- `chrome.storage.sync` is used for settings/preferences (for example toggles, colors, and UI options).
- `chrome.storage.local` is used for local wallpaper image data and related local metadata.

Data is used only for the extension's single purpose described above.

## Scope

TeamTweaker runs only on:

- `https://teams.microsoft.com/*`
- `https://teams.live.com/*`
